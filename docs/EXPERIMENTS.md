# Experiment map

1. `operator_construction_demo.py`: exports the finite Jacobi matrix and coefficient sequence.
2. `generator_construction_demo.py`: verifies unitary evolution generated by the same matrix.
3. `zero_alignment_demo.py`: demonstrates ordinate reconstruction.
4. `control_survival_demo.py`: compares signal with adversarial controls.
5. Stable global validation: precision/radius/direct derivative/Carleman/perturbation tests.
6. GUE audits: raw, stable-bulk, frontier, and preregistered holdout tests.
7. `generate_plots.py`: regenerates all included figures from CSV data.
